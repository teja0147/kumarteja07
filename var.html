Variables and Operators in JavaScript

Introduction to Variables

In JavaScript, variables are used to store data that can be referenced and manipulated throughout the program. You declare variables using the var, let, or const keyword.

Example:


        // Declaring variables
        var myVariable; // Declaring a variable without initializing it
        let age = 25;   // Declaring and initializing a variable using let
        const PI = 3.14; // Declaring and initializing a constant variable using const

Data Types

JavaScript supports several data types:

  • Number: Represents numeric data, including integers, floats, and NaN (Not-a-Number).
  • String: Represents text data enclosed within single ('') or double ("") quotes.
  • Boolean: Represents true or false values.
  • Null: Represents an empty value.
  • Undefined: Represents a variable that has been declared but not initialized.
  • Object: Represents a collection of key-value pairs.
  • Symbol: Represents unique identifiers.

Operators

JavaScript provides various operators for performing operations on variables and values:

  • Arithmetic Operators: Used to perform arithmetic operations like addition, subtraction, multiplication, division, and modulus.
  • Assignment Operators: Used to assign values to variables.
  • Comparison Operators: Used to compare values.
  • Logical Operators: Used to combine or modify conditions.

Variable Scope

Variables in JavaScript have function-level scope with the var keyword and block-level scope with the let and const keywords. Variables declared with var are function-scoped, while variables declared with let and const are block-scoped.

Conclusion

Understanding variables and operators is fundamental in JavaScript programming. They provide the building blocks for creating dynamic and interactive web applications.
